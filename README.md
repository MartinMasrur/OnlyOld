# 🕰️ OnlyOld

> A social network with no new content — because the past is all we need.

---

## 🌟 The Idea

In a world obsessed with the "new", **OnlyOld** is a rebellion.  
A digital time capsule where you can only share, view, and interact with **content from the past**.

No fresh uploads. No "what's happening now".  
Just memories, archives, and the beauty of what already exists.

---

## 🧩 Why OnlyOld?

- 🔁 **Rediscover** old photos, posts, and moments
- 🧠 **Curate** your digital history, not your daily life
- 🕊️ **Escape** the pressure of constant content creation
- 📦 **Preserve** what truly matters — without the noise

---

## 🚧 Current Status

This project is in early development.  
If you're nostalgic about the internet of 2010s or just tired of infinite scrolling — you're in the right place.

---

## 🛠️ Tech Stack (Planned)

- **Frontend**: HTML, CSS, JavaScript (or React)
- **Backend**: Node.js / Python (Django/Flask)
- **Database**: PostgreSQL / MongoDB
- **Hosting**: GitHub Pages (for now)
